## Projeto Django
### Descrição 
O sistema de e-commerce permite que os usuários naveguem por produtos, visualizando informações como nome, descrição e preço. Os clientes podem adicionar produtos ao carrinho e finalizar pedidos. Além disso os usuários têm a opção de avaliar produtos. O sistema também mantém um controle de estoque, evitando que um produto sem estoque seja vendido.
